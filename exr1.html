วันแรกของการเรียนรู้ HTML

ทำไม HTML ถึงสำคัญ

HTML เป็นโครงสร้างหลักของเว็บไซต์ทุกแห่งบนโลก ถ้าเปรียบกับการสร้างบ้าน HTML ก็คือเสาเข็มและโครงเหล็กนั่นเอง

การเข้าใจ HTML จะช่วยให้เราต่อยอดไปเรียนรู้การแต่งหน้าตาเว็บด้วย CSS และทำให้เว็บทำงานได้ด้วย JavaScript

ขั้นตอนการเรียนรู้เบื้องต้น

  1. ศึกษาโครงสร้างพื้นฐานของ Tag
  2. ลองสร้างหน้าเว็บที่มีแต่ข้อความ
  3. ฝึกใส่รูปภาพและลิงก์เชื่อมโยง

สิ่งที่เราจะได้เรียนรู้

ในบทเรียนนี้เราจะได้ฝึกใช้ Tag สำคัญๆ เช่น การทำหัวข้อ การใส่ย่อหน้า และการจัดการรูปภาพเพื่อให้เว็บดูน่าสนใจ

  • MDN Web Docs - คู่มือสำหรับนักพัฒนา
  • YouTube ช่องสอนเขียนโปรแกรมต่างๆ
